# Start Here

## Quick Practice Mode

Open `index.html`.

The 50 built-in cards, shuffle button, flip card, and history button will work.

## Full Mode With Translation

Use this if you want to add new English sentences and have Mandarin and German translated automatically.

On Windows:

1. Double-click `start-windows.bat`.
2. Your browser should open `http://localhost:8000`.

If the start file says Python is missing, install Python from:

https://www.python.org/downloads/

Then double-click `start-windows.bat` again.
