# Language Flip Cards

A small web app for practicing English sentences with Mandarin and German translations.

## Share It Online With Vercel

1. Go to https://vercel.com and sign in.
2. Create a new project.
3. Upload this `card-translator` folder.
4. Deploy it.
5. Send your friend the Vercel link.

The page and the translation backend are both included in this folder.

## Run It On Your Computer

If Python is installed, open this folder in a terminal and run:

```bash
python server.py
```

Then open:

```text
http://localhost:8000
```
